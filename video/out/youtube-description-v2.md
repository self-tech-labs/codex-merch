## Description

Codex Merch is an open-source, hackable pipeline from a trusted trend signal to
real, production-ready merch—not just a generated picture.

Try the public, non-purchasable Preview:
https://codex-merch-git-codex-build-week-weekly-studio-ritsl.vercel.app

The demo starts with the Solward Index output: a catalog entry, four customer
views, six provider-sized print panels, artifact hashes, and a public rights
record. It then reduces the system to five moves—Signal, Direct, Build, Prove,
Release—and shows the four repository seams a team can replace: source,
judgment contract, physical product, and outcome.

The commercial thesis is broader than this storefront. Zara and Shein are
examples of the speed-to-product problem; Richemont and LVMH are examples of a
controlled fashion-R&D problem where sources, models, product formats, and
approval policies need to remain independently testable. These companies are
market examples only. No affiliation, endorsement, customer relationship, or
use of proprietary data is claimed.

AI disclosure: GPT-5.6 produces bounded structured trend and art-direction
judgments and critiques actual rendered garments. Deterministic code owns
provenance, rights checks, exact six-panel composition, prepress, hashes, state,
and release gates. Narration is an AI-generated voice synthesized with OpenAI
gpt-audio-1.5.

This is an independent OpenAI Build Week project. Checkout is disabled, and no
payment, provider sync, publication, or fulfillment action is demonstrated.
