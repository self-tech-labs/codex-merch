# Merch References

This folder contains production and art-direction references used by the merch
pipeline.

- `printful/aop-cotton-sweatshirt/templates`: Printful All-Over Print Unisex
  Cotton Sweatshirt panel templates. Final production files must match these
  canvas dimensions and must not include guide layers.
- `art-direction/supplyco-screenshots`: local screenshots used as aesthetic
  references for the adversarial art director validator. They are references
  only, not source artwork to copy.
